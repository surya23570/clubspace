# ClubSpace -- UI/UX Design Specification

## 1. Design Philosophy

The interface follows a modern social-feed layout with soft shadows,
rounded cards, and a clean creator-focused aesthetic. The design
prioritizes visual clarity and fast content scanning.

## 2. Layout Structure

### 2.1 Three-Column Desktop Layout

-   Left Sidebar (Navigation)
-   Center Feed (Primary Content)
-   Right Sidebar (Discovery & Suggestions)

------------------------------------------------------------------------

## 3. Left Sidebar

### Profile Section

-   Circular avatar
-   Username
-   Handle text
-   Soft background

### Navigation Menu

Items: - News Feed (active highlight) - Messages - Forums - Friends -
Media - Settings

Style: - Rounded pill active state - Minimal monochrome icons - Vertical
spacing

### Download App Card

-   Dashed border container
-   Gradient icon
-   Soft rounded corners

------------------------------------------------------------------------

## 4. Center Feed

### Feed Header

-   Title: "Feeds"
-   Filter tabs: Recents \| Friends \| Popular
-   Subtle typography hierarchy

### Post Card Design

#### Card Container

-   Large rounded corners
-   Soft shadow
-   Light pastel background

#### Post Header

-   Avatar (small circle)
-   Author name (bold)
-   Timestamp (muted)
-   Overflow menu (3 dots)

#### Post Content

-   Text paragraph
-   Multi-image grid (rounded)
-   Consistent spacing

#### Post Footer

-   View count
-   Like
-   Comment
-   Reaction bubble (floating)

Interaction: - Hover elevation - Smooth transitions

------------------------------------------------------------------------

## 5. Right Sidebar

### Stories Section

-   Horizontal story cards
-   Rounded rectangles
-   Small avatar overlay

### Suggestions

-   User avatar
-   Name
-   Follow button (pill)

### Recommendations

Category bubbles: - UI/UX - Music - Cooking - Hiking

Style: - Soft gradient circles - Minimal labels

------------------------------------------------------------------------

## 6. Bottom Composer Bar

-   Floating input
-   Rounded full-width container
-   File / Image / Location actions
-   Public dropdown
-   Send button (dark pill)

------------------------------------------------------------------------

## 7. Visual System

### Colors

-   Soft pastel backgrounds
-   Neutral grays
-   Accent gradient highlights
-   Dark primary buttons

### Typography

-   Clean sans-serif
-   Bold for names
-   Muted metadata
-   Clear hierarchy

### Spacing

-   Generous padding
-   Card separation
-   Grid consistency

------------------------------------------------------------------------

## 8. Responsiveness

### Tablet

-   Collapse right sidebar
-   Maintain feed priority

### Mobile

-   Single column
-   Bottom navigation
-   Sticky composer

------------------------------------------------------------------------

## 9. Accessibility

-   Proper contrast
-   Labelled inputs
-   Keyboard navigation
-   Focus states
