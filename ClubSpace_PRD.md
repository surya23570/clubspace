# ClubSpace -- Product Requirements Document (PRD)

## 1. Product Overview

ClubSpace is a private, college-only creative talent platform designed
to centralize, showcase, and evaluate student creative work. The
platform replaces scattered sharing across WhatsApp, Instagram, and
personal drives with a structured, verifiable system.

## 2. Goals

-   Provide a single source of truth for student creative work
-   Enable skill-based feedback instead of vanity likes
-   Maintain monthly visibility via live feed
-   Preserve long-term student portfolios
-   Provide fair monthly recognition

## 3. Target Users

-   Students (Members)
-   Mentors / Club Leads
-   Admin / Faculty

## 4. Core Features (MVP)

### 4.1 Authentication

-   Email/password login
-   Role-based access
-   Session persistence

### 4.2 Content Upload

-   Media upload (image/video/audio)
-   Category tagging
-   Purpose tagging
-   Cloudinary storage

### 4.3 Monthly Live Feed

-   Shows only current month posts
-   Sorted by newest
-   Responsive grid

### 4.4 Permanent Portfolio

-   User profile gallery
-   Upload history
-   Feedback summary
-   Export-ready

### 4.5 Skill-Based Feedback

Reaction types: - Creative Idea - Strong Composition - Editing Quality -
Emotional Impact - Needs Improvement

Rules: - One reaction per user per post - Mentors weighted higher

### 4.6 Monthly Recognition

Leaderboard based on: score = uploads \* 2 + feedback_received +
mentor_bonus

Awards: - Best Photographer - Best Designer - Most Improved - Most
Active - Rising Talent

### 4.7 Admin Dashboard

-   Total uploads
-   Active users
-   Category distribution
-   Monthly analytics

## 5. Non-Functional Requirements

-   Mobile responsive
-   Fast media loading
-   Secure authentication
-   Scalable architecture
-   Clean modern UI

## 6. Success Metrics

-   Weekly active users
-   Upload frequency
-   Feedback engagement
-   Mentor participation
